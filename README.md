# 第二種電気工事士 学習アプリ

第二種電気工事士の筆記試験合格を目的とした、ブラウザで動くシングルページ学習ツールです。

## 機能

| タブ | 内容 |
|------|------|
| ホーム | 学習進捗（今日の問題数・正答率）とカテゴリ選択 |
| 過去問 | 4択クイズ（カテゴリ別 / ランダム）+ 解説表示 |
| 公式・計算 | オームの法則・電力・合成抵抗・力率など主要公式一覧 |
| 用語暗記 | タップで定義を開閉するフラッシュカード式暗記 |
| 配線図 | SVG回路図つき配線問題（draw.io MCP連携予定） |

## ファイル構成

```
denki/
├── index.html              # エントリーポイント
├── .mcp.json               # draw.io MCP設定
├── README.md
└── src/
    ├── css/
    │   └── style.css       # 全スタイル（CSS変数定義含む）
    ├── data/
    │   ├── questions.js    # 問題データ（8問）
    │   └── vocab.js        # 用語データ（10語）
    └── js/
        ├── quiz.js         # 過去問ロジック
        ├── vocab.js        # 用語暗記ロジック
        ├── wiring.js       # 配線図ロジック
        └── app.js          # タブ切替・初期化
```

## 使い方

ビルド不要。`index.html` をブラウザで直接開くだけで動作します。

```bash
# ローカルサーバーを立てる場合（Python）
python -m http.server 8080
```

## draw.io MCP（配線図の拡張）

配線図タブを draw.io でインタラクティブに編集するには Node.js が必要です。

1. [Node.js](https://nodejs.org/) をインストール
2. Claude Code を再起動（`.mcp.json` が自動読み込みされる）
3. 「draw.io で3路スイッチの複線図を作って」と指示する

## 問題データの追加

`src/data/questions.js` の `questions` 配列にオブジェクトを追加してください。

```js
{
  category: '電気理論',          // 電気理論 / 配線設計 / 材料・工具 / 法令
  text: '問題文',
  choices: ['選択肢1', '選択肢2', '選択肢3', '選択肢4'],
  answer: 0,                     // 正解のインデックス（0始まり）
  explanation: '解説文'
}
```

## 用語データの追加

`src/data/vocab.js` の `vocabData` 配列に追加してください。

```js
{ term: '用語名', def: '定義・説明文' }
```
